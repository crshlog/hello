The Web Developer

Apa sih Web Developer itu?

Web Developer adalah perancang sebuah website / www yang merancang sebuah website dari awal sekali, menciptakan coding dsb, atau bisa juga seorang Web Developer mengembangkan sebuah website yang sudha ada. Web Developer bisa bekerja secara terikat dengan perusahaan, atau menjadi pekerja lepas (freelancer). Jika ingin menjadi seorang Web Developer yang ahli, kita harus memiliki pengetahuan dasar seputar coding, kemampuan desain multimedia, dan manajemen. Lalu untuk pekerjaannya, ada 3 tugas utama yaitu:

  1. Front End Developer

    2. Front End Developer adalah tampilan utama sebuah website. Mulai dari desainnya, tulisannya, warna, dsb. Jadi, Front End Developer membuat tampilan website yang enak dipandang oleh user, seperti bagian user interface dan user experiencenya. Seorang Front End Developer harus menguasai beberapa bahasa pemrograman, seperti HTML, CSS, dan Javascript. HTML berguna untuk merangkai struktur website, CSS berguna untuk membuat tampilan dari website, sedangkan Javascript berguna untuk menjalankan website. Bisa dibilang, javascript adalah roda yang menggerakkan website tersebut, CSS membuat tampilan website yang indah, dan HTML menjadi dasar dari sebuah website.

  2. Back End Developer

    3. Back End Developer mengurus server, database, dan aplikasi. Jadi Back end developer mengurus data user, penyimpanan data user, dll. Seorang Back End Developer harus menguasai beberapa bahasa pemrograman juga, seperti PHP, Python, MySQL, dll. memang lebih banyak dari Front End, karena pekerjaan Back End Developer lebih ke arah menjaga data, database, dsb

  3. Full Stack Developer

    4. Full Stack Developer adalah seseorang yang mengurus baik Front dan Back End website. Jadi dia harus menguasai semua bahasa pemrograman yang dibutuhkan untuk mengurus Front dan Back End website. Tanggung jawab seorang full stack juga jadi lebih banyak dibanding Front end maupun back end
